# exitTime

Плагин **exitTime.asi** для **GTA San Andreas** с **SA:MP** (и одиночной игры без `samp.dll`): настраивает время выхода из игры.

## Установка

- Положите **exitTime.asi** рядом с игрой или в **modloader\\asi** (или папку scripts modloader).
- Рядом с ASI при первом запуске сам создаётся **exitTime.ini** с короткими подсказками в строках, начинающихся с **`;`**.

## Настройки (файл exitTime.ini, секция `[Settings]`)

- **Time in milliseconds** — время в миллисекундах для SA:MP (выгрузка и `/quit` / `/q`) и для ограничения длинных пауз при выходе в **GTA 1.0 US** (см. комментарии в ini). Значение **0** может вести себя нестабильно. Старый способ: **-1** / **-2** — мгновенный выход процесса (см. комментарии в ini).
- **CGame shutdown exit** — **0** обычный выход, **1** / **2** — принудительно закрыть игру при начале выхода (см. комментарии в ini).
- **SAMP unload exit** — **0** как обычно (патчатся только числа времени в `samp.dll`, **без** перехвата `GetTickCount` — так стабильнее выход из меню «Выйти»). **1** / **2** — принудительный выход в цепочке выгрузки samp (см. комментарии в ini; для **R2** через samp недоступно). Перехват `GetTickCount` также включается при **Time in milliseconds** **-1** / **-2** (устаревший способ).

## Важно

- Поддерживаются только перечисленные в коде сборки **samp.dll** (по entry point). Неизвестная версия — патчи к samp не применяются.
- Часть для GTA (ограничение пауз при выходе) работает только на **gta_sa.exe 1.0 US** (compact или Hoodlum).
