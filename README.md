# FastExit

Плагин **FastExit.asi** для **GTA San Andreas** с **SA:MP** (и одиночной игры без `samp.dll`): один режим выхода **0 / 1 / 2** для меню «Выйти» и для цепочки выгрузки samp после **`/q`** и **`/quit`**.

## Установка

- Положите **FastExit.asi** рядом с игрой или в **modloader\\asi** (или папку scripts modloader).
- Рядом с ASI при первом запуске сам создаётся **FastExit.ini** с короткими подсказками в строках, начинающихся с **`;`**.

## Поддерживаемые версии `samp.dll`

Детект по **AddressOfEntryPoint** (RVA). В коде заведены такие метки (см. `kSupportedVersions` в `source/main.cpp`):

| Метка в коде | Примечание |
|--------------|------------|
| **R1** | |
| **R2** | для режимов **1/2** по чату нет RVA подмены вызова — только меню через GTA |
| **R3** | |
| **R3-1** | |
| **R4** | |
| **R4-2** | |
| **R5-1** | клиент **R5-2** с тем же entry point распознаётся как эта строка |
| **DL-R1** | клиент **DL-R1-2** с тем же entry point — аналогично |

Иная сборка `samp.dll` — подмена на пути выгрузки **не** применяется.

## Настройки (файл FastExit.ini, секция `[Settings]`)

- **Exit mode** — **0** обычный выход; **1** — `ExitProcess`; **2** — `TerminateProcess`.
  - **Меню GTA «Выйти»:** при **1** или **2** ставится MinHook на **`CGame::Shutdown`** (только **gta_sa.exe 1.0 US**, compact или Hoodlum). У части конфигураций с **SA:MP** хук на Shutdown может давать зависание при спавне — тогда оставляйте **0** или проверяйте без плагина.
  - **`/q` / `/quit`:** при **1** или **2** подменяется **один** вызов на пути выгрузки `samp.dll` у перечисленных версий (кроме **R2**, см. таблицу).

## Важно

- Инициализация: при **SA:MP** сначала правки к известной **samp.dll**, затем хук **GTA 1.0 US** (порядок важен).
- Часть для GTA (`CGame::Shutdown`) работает только на **gta_sa.exe 1.0 US** (compact или Hoodlum).
